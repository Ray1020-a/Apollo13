# 貼皮資源目錄 Textures

將 PNG 貼圖放入對應子資料夾，檔名必須與下表一致。  
遊戲啟動時會自動載入；若檔案不存在則使用程式內建的純色備援材質。

## 太空船 Spacecraft (`spacecraft/`)

| 檔名 | 尺寸 (寬×高) | 用途 | UV 說明 |
|------|-------------|------|---------|
| `body.png` | **1024 × 2048** | 主艙體（圓柱） | 橫向環繞一圈、縱向為艙體高度 |
| `shield.png` | **512 × 512** | 防熱盾（錐體底部） | 從頂點展開的扇形展開圖 |
| `tunnel.png` | **512 × 512** | 對接隧道（小圓柱） | 環繞展開 |
| `panel.png` | **1024 × 512** | 太陽能板（左右各一，共用） | 平面貼圖 |
| `window.png` | **256 × 256** | 舷窗圓形玻璃 | 圓形 UV |
| `cabin-interior.png` | **2048 × 1024** | 艙內全景（球面內側） | 等距柱狀投影 Equirectangular |

## 艙內儀表板 Cockpit (`cockpit/`)

| 檔名 | 尺寸 (寬×高) | 用途 | 說明 |
|------|-------------|------|------|
| `instrument-panel.png` | **2048 × 1024** | 儀表板面板底圖 | 3D 面板 + HTML HUD 共用；中央留空給動態數值 |
| `button.png` | **128 × 64** | 按鈕貼皮 | 設備開關與視角切換按鈕 |
| `gauge-face.png` | **256 × 256** | 儀表區塊底圖 | 各 gauge 區塊背景 |
| `frame.png` | **512 × 1024** | 駕駛艙窗框／柱子 | 金屬結構貼皮 |
| `glass.png` | **1024 × 768** | 擋風玻璃 | 透明玻璃，建議含 Alpha |

### 儀表板貼皮安全區（`instrument-panel.png`）

底部橫條佈局（寬 2048，建議由上到下分三帶）：

```
┌─────────────────────────────────────────┐  ← 上 55%：讀數區（ETA / PWR / O2…）
│  可繪製錶面外框，中央留空給動態數字       │
├─────────────────────────────────────────┤  ← 中 15%：狀態列（相位 / 警告）
│  建議暗色底或留白                         │
├─────────────────────────────────────────┤  ← 下 30%：按鈕列
│  建議留白或按鈕槽位暗色底                 │
└─────────────────────────────────────────┘
```

## 星空 Stars (`stars/`)

| 檔名 | 尺寸 (寬×高) | 用途 | UV 說明 |
|------|-------------|------|---------|
| `skybox.png` | **4096 × 2048** | 星空背景球（內側） | 等距柱狀全景 |
| `particle.png` | **64 × 64** | O2 粒子噴射 sprite | 正方形，建議透明背景 |

## 製作建議

- 格式：PNG，建議含 Alpha 通道（粒子、舷窗、大氣層）
- 色彩空間：sRGB
- `body.png` 上下邊緣可留 1–2% 空白，避免 UV 接縫露餡
- `cabin-interior.png` 正中央為前方舷窗方向（+Z），左右為艙壁
- `skybox.png` 可從 NASA 星空圖或自製星點圖縮放至 4096×2048
