# 背景音樂 Audio

將 MP3 檔放入此資料夾，遊戲啟動後會**無限循環**播放。

## 檔案

| 檔名 | 用途 |
|------|------|
| `bgm.mp3` | 背景音樂（預設檔名，必填才會有聲音） |

## 使用方式

1. 把你的音樂重新命名為 `bgm.mp3`
2. 放到 `public/audio/bgm.mp3`
3. 重新整理頁面

## 注意

- 格式：**MP3**
- 瀏覽器可能阻擋自動播放；若沒聲音，**點一下畫面或按任意鍵**即可開始
- 音量預設 45%，可在 `src/ui/bgm.ts` 調整 `BGM_VOLUME`

## 目錄結構

```
public/audio/
├── README.md   ← 本說明
└── bgm.mp3     ← 你的背景音樂
```
